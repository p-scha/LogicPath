# LogicPath

LogicPath is a gamified educational web application that focuses on learning formal and informal logic.
It blends interactive gameplay with educational content to make logic engaging, accessible, and fun.

LogicPath is a collaborative group project currently under active development.

Team Website: https://github.com/p-scha/LogicPath-Team-Website

# Installation

Install node.js and npm from the [nodejs official website](https://nodejs.org/en/download).

### Commands
```
# Navigate to the directory you want to store project and clone repo
git clone https://github.com/p-scha/LogicPath.git

# Navigate to project directory
cd ./logic-path

# Install dependencies
npm install

# Start development server
npm run dev
```
